# Основные команды

## start

Запуск среды исполнения контейнеров (виртуальной машины).

```shell
platformctl start
```

!!! note

    На данный момент необходим и поддерживается только для `minikube`.

## stop

Остановка среды исполнения контейнеров (виртуальной машины).

```shell
platformctl stop
```

!!! note

    На данный момент необходим и поддерживается только для `minikube`.
